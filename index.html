PDF FIT - PDF Size Reducer

Welcome to PDF FIT

A user-friendly tool to compress PDF files efficiently.

How to Use

Follow these steps to reduce your PDF file size:

  1. Clone the repository.
  2. Install dependencies using the provided script.
  3. Run the PDF Size Reducer script.
  4. Follow on-screen instructions to reduce PDF size.

Future Features

Potential upcoming features for PDF FIT:

  • Improved user interface.
  • Batch processing of multiple PDF files.
  • Integration with cloud storage services.
